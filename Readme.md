# Neovim configuration as a IDE

### üìù Completion \& Snippets

- [LuaSnip](https://github.com/L3MON4D3/LuaSnip) - Snippet engine for Neovim written in Lua.
- [cmp-nvim-lsp](https://github.com/hrsh7th/cmp-nvim-lsp) - LSP source for nvim-cmp.
- [cmp-spell](https://github.com/f3fora/cmp-spell) - Spell source for nvim-cmp.
- [cmp_luasnip](https://github.com/saadparwaiz1/cmp_luasnip) - LuaSnip source for nvim-cmp.

- [nvim-web-devicons](https://github.com/nvim-tree/nvim-web-devicons) - Filetype icons for Neovim plugins.
- [nvim-colorizer.lua](https://github.com/norcalli/nvim-colorizer.lua) - Fast color highlighter.
